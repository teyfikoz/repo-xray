## What is this?

FastAPI is a modern, high-performance Python web framework for building APIs with automatic OpenAPI documentation. It targets Python developers who want type-safe, async-first REST APIs without boilerplate. The framework generates interactive Swagger UI and ReDoc docs from type hints alone.

## Tech Stack

- `Python 3.8+`
- `Starlette`
- `Pydantic v2`
- `Uvicorn / Hypercorn`
- `OpenAPI 3.x`
- `Anyio`
- `pytest / httpx (testing)`

## Architecture

FastAPI is a thin wrapper around Starlette, adding dependency injection, Pydantic-based request/response validation, and automatic schema generation. The framework follows a decorator-based routing pattern (Flask-style) with async-first handlers and a powerful `Depends()` system for shared logic.

## Key Files

- **`fastapi/applications.py`** — The main `FastAPI` class; subclasses Starlette and bootstraps routing, middleware, and OpenAPI generation
- **`fastapi/routing.py`** — `APIRouter` and route registration; converts type hints to Pydantic models at startup
- **`fastapi/dependencies/utils.py`** — Dependency injection resolver; handles `Depends()` chain resolution at request time
- **`fastapi/openapi/utils.py`** — OpenAPI schema generation from type hints, docstrings, and Pydantic models
- **`fastapi/security/`** — OAuth2, HTTP Basic, API key, and Bearer token security schemes

## Learning Insights

**1.** FastAPI resolves the entire dependency graph at *import time*, not at request time — this means startup is slow but each request is fast, a deliberate trade-off.

**2.** The `response_model` parameter does not change what your function returns; it runs the return value through a Pydantic model to strip fields and validate output — a subtle but important distinction.

**3.** Background tasks (`BackgroundTasks`) run after the HTTP response is sent, making them useful for fire-and-forget operations like sending emails without blocking the client.

## Quick Start Guide

1. `pip install fastapi uvicorn`
2. Create `main.py` with `from fastapi import FastAPI; app = FastAPI()`
3. Add a route: `@app.get("/") def root(): return {"message": "Hello World"}`
4. Run: `uvicorn main:app --reload`
5. Open http://127.0.0.1:8000/docs for interactive API explorer
6. Add Pydantic models for request bodies: `class Item(BaseModel): name: str; price: float`
7. Use `@app.post("/items/")` with `item: Item` as a parameter for auto-validation
